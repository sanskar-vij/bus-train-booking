# Bus & Train Booking (starter)

Project structure created by `create_bus_train_booking.py`.

## Run locally

1. (Optional) Create virtualenv:
   - Linux/macOS:
     ```
     python -m venv venv
     source venv/bin/activate
     ```
   - Windows (PowerShell):
     ```
     python -m venv venv
     venv\Scripts\Activate.ps1
     ```

2. Install deps:
   ```
   pip install -r requirements.txt
   ```

3. Run the app:
   ```
   python app.py
   ```

4. Open http://127.0.0.1:5000

## Deploy
- A simple `Procfile` is included for hosting providers like Heroku / Railway (adjust as needed).
