# Bus & Train Ticket Booking System

A Flask-based web application for booking bus and train tickets.

## Features
- User registration & login
- Search buses/trains
- Ticket booking
- Database integration (SQLite)

## Tech Stack
- Python
- Flask
- HTML, CSS
- SQLite

## How to Run
1. Install dependencies  
   `pip install -r requirements.txt`
2. Run the app  
   `python app.py`
